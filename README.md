# ScanPhotos-X X图片浏览器
本图片浏览器支持网页上图片的自动扫描加载，支持body内所有图片或者指定区域内所有图片的自动扫描加载。并且当图片发生添加或删除时可动态更新。同时支持对图片的操作处理包括鼠标滚轮放大缩小，图片旋转，图片的鼠标拖拽移动，图片下载，图片切换，全屏展示等功能
,支持在一个网页实现多个实例。
